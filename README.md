# Dead Run

A 2D infinite side scroller made using [Flame](https://flame-engine.org/) engine.

![Jump](branding/jump.gif)

## Build steps

```bash
# Para clonar el proyecto
$ git clone https://github.com/Dnovan/Dead run

# Acceso
$ cd Dead run

# Instalación de dependencias
$ flutter pub get

# Ejecutar el proyecto 
$ flutter run

# Esperas la ejecución en proyecto en web o movil.
```

## Supported platforms

- Android
- iOS
- Windows (el audio se bugea)
